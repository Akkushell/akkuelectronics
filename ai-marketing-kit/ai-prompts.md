# Reusable AI Prompts

## 1) Daily Post Generator
Create 5 social media posts for Akku Electronics (console repair business in Nagpur).
Platforms: Facebook, Instagram, X.
Tone: friendly, expert, local, trustworthy.
Services: PS5/PS4/PS3/Xbox repairs, HDMI, no power, cleaning, controller repair.
For each post include: hook, caption, CTA to WhatsApp, and 8 hashtags.

## 2) Reel Script Prompt
Write a 20-second Instagram reel script for [repair topic].
Include:
- 1-line hook
- 3 short scene instructions
- on-screen text
- final CTA “DM for quote”
Keep language simple and high-energy.

## 3) Carousel Prompt
Create a 6-slide educational carousel on [topic].
Slide 1 = strong hook.
Slides 2-5 = practical tips.
Slide 6 = CTA to WhatsApp + website.
Also provide caption and hashtags.

## 4) X Thread Prompt
Write a 7-tweet thread on [topic] for gamers.
Style: concise, helpful, no fluff.
Include one CTA at the end to https://akkuelectronics.in and WhatsApp.

## 5) Ad Copy Variants Prompt
Generate 10 ad copy variants for [service].
Audience: gamers in Nagpur.
Format each with: primary text, headline, CTA, emotional angle.

## 6) Comment Reply Prompt
Generate 20 short comment replies for social media.
Categories: pricing query, timeline query, appreciation, urgent issue, spam-safe polite response.

## 7) Offer Post Prompt
Create weekend offer post for console diagnostics.
Need 3 versions:
- urgency style
- trust style
- budget style
with CTA to WhatsApp.
